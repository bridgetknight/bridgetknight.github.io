PiCar: Autonomous RC Car
An autonomous remote-controlled car built with a Raspberry Pi and various sensors (LiDAR, cameras), featuring obstacle detection and path planning capabilities.
Python Raspberry Pi Computer Vision AI/ML

This project involved assembling the Raspberry PiCar kit and programming it using Python to achieve autonomous, self-driving functionality. The implementation utilized computer vision and image processing techniques, leveraging the PiCamera module and methods like edge detection. (1) An intermediate image of the car's vision feed after performing image processing to detect the track. (2) An image of the car's final path, plotted using OpenCV and numpy. (3) The car on its calibrating loop track.

MBTA Trip Planner
A full-stack route planner for Boston's MBTA transit system, using Dijkstra's algorithm on a custom station graph with real-time API data.
MERN Stack UI/UX Algorithms REST APIs

Built with the MERN stack and previously hosted on Render. Implements Dijkstra's algorithm on a hand-designed graph of MBTA stations to find optimal routes, pulling live departure data from the MBTA API and location search from the Radar API.
